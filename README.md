# 📊 EMI Calculator

A simple, responsive **EMI (Equated Monthly Installment) Calculator** that helps users quickly calculate their monthly loan payments based on principal amount, interest rate, and tenure.

---

## 🚀 Live Demo

👉 https://emi-calculator-lake-psi.vercel.app/  

---

## 📌 Features

- 💰 Calculate EMI instantly  
- 📱 Fully responsive (mobile-friendly)  
- ⚡ Fast and lightweight  
- 🎯 Clean and user-friendly UI  
- 🔗 External link integration (“Built for Digital Heroes”)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 📷 How It Works

1. Enter loan amount  
2. Enter interest rate  
3. Enter loan tenure  
4. Click calculate  
5. Get your monthly EMI instantly  

---

## 🎯 Use Case

This tool helps users estimate loan EMIs for better financial planning.  
I’ve personally used similar calculators to understand loan affordability before making financial decisions.
